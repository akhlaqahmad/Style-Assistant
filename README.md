# StylistA – Personal Styling App

StylistA is a personal styling and wardrobe companion mobile app built with Expo React Native. It helps users understand their body proportions and colour tones, digitise their wardrobe, generate daily outfits, plan travel capsules, and book professional stylists.

## 🚀 Quick Start

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npx expo start
    ```
    or
    ```bash
    npm run expo:dev
    ```

3.  **Run on iOS/Android**:
    -   Download the Expo Go app.
    -   Scan the QR code shown in the terminal.

For more detailed setup instructions, see [SETUP.md](docs/SETUP.md).

## ✨ Key Features

-   **Onboarding Flow**: Personalized onboarding to determine style preferences.
-   **Tone Analysis**: AI-driven color palette analysis based on physical attributes.
-   **Wardrobe Management**: Digitize your closet and manage items.
-   **Outfit Generator**: Get daily outfit suggestions.
-   **Travel Planner**: Plan outfits for trips.
-   **Stylist Marketplace**: Book professional stylists.
-   **Body Scanning**: Capture measurements for a proportional avatar.
-   **Virtual Try-On**: Visualize outfits on your custom avatar.
